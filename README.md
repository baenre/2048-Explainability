# 2048-Explainability

To play the game, download the repository as a .zip file and open index.html in the folder. The rest of the steps are described on the game screen. 
It’s important that each policy selected from the dropdown menu is played three times. There is no order you have to follow when it comes to choosing a policy and if you want to play more than the required rounds, feel free to do so! But please note, that game policies should not be changed during a game, as this will restart the game. Therefore, only change them once the game has been won or lost. After playing the game three times with one policy, click “Download”, repeat this with the other two policies and then send the three files back to me. 

Thank you and have fun!
